# Linux 1日目 (2026/05/02)

## 学習項目
- **Linuxとは何か？**
  - コンピューターを動かすための基本ソフト（OS）の一種。
- **Linuxの特徴**
  - オープンソース：誰でも無料で利用・改変が可能。
  - 高い信頼性：世界中のサーバーやシステムで採用されている。
  - 豊富なソフトウェア：高品質なツールが多数利用できる。

## なぜLinuxを学ぶのか？
- **サーバー・組み込み領域の標準**: 開発現場でマシンを操作する際の必須スキル。
- **技術が枯れている**: 長年使われ続けており、安定した知識として一生使える。
- **Java/Spring Bootとの関連**: 作成したアプリを動かす本番環境の多くがLinuxであるため。

## Linuxの構成要素
- **カーネル (Kernel)**: OSの中核。ハードウェアの制御を司る「本体」。
- **ディストリビューション**: カーネルに様々なソフトを組み合わせ、使いやすくパッケージ化したもの（例：Rocky Linux）。

## シェルとインターフェース
- **シェル (Shell)**: ユーザーの命令をカーネルに伝える「通訳」ソフト。
- **CLI (Command Line Interface)**: キーボードでコマンドを入力し、文字で結果を得る形式。
- **GUI (Graphical User Interface)**: マウスやアイコンで直感的に操作する形式。

### 役割イメージ
- **ユーザー** → **シェル** → **カーネル**
- キーボード入力をシェルが受け取り、カーネルがわかる言葉に翻訳して伝える。
