## コマンド備忘録
- `pwd`: 今いる場所（カレントディレクトリ）を表示。
- `cd /`: ルートディレクトリ（最上層）へ移動。
- `ls`: ディレクトリの中身を表示（`-l`で詳細、`-a`で隠しファイルを表示）。
- `clear`: ターミナルの表示を綺麗にする（`Ctrl + l`でも可）。
- `mkdir`: 新しいディレクトリを作成。
- `rmdir`: 空のディレクトリを削除。

## ショートカットキーまとめ
### カーソル移動
- `Ctrl + f`: 1文字次へ移動。
- `Ctrl + b`: 1文字前へ移動。
- `Ctrl + a`: 行の先頭に移動。
- `Ctrl + e`: 行の最後に移動。
- `esc + f`: 1単語次へ移動。
- `esc + b`: 1単語前へ移動。

### 文字列の編集
- `Ctrl + h`: カーソルの左を1文字削除。
- `Ctrl + d`: カーソル部分を1文字削除。
- `Ctrl + w`: カーソル位置の単語を削除。
- `Ctrl + u`: カーソル位置から行頭まで削除。
- `Ctrl + k`: カーソル位置から行末まで削除。
- `Ctrl + y`: 最後に削除した内容を挿入（ヤンク）。

### トラブル・画面操作
- `Ctrl + c`: 実行中のコマンドを強制終了。
- `Ctrl + s`: 画面表示のロック（入力が反映されなくなる）。
- `Ctrl + q`: 画面表示のロック解除。

## Linuxのディレクトリ構造
- **ディレクトリツリー**: ルートディレクトリ `/` を頂点とした階層構造。Linuxは一つのツリーしか持たない。
- **/bin**: 重要度の高い基本コマンドの実行ファイルを格納。
- **/dev**: デバイスファイルを格納。
- **/etc**: システムやアプリの設定ファイルを格納。
- **/home**: ユーザーごとの個人ディレクトリ。
- **/sbin**: システム管理者用コマンドを格納。
- **/tmp**: 一時的なファイルを格納（アプリの実行用など）。
- **/usr**: アプリケーション本体や関連ファイルを格納。
- **/var**: ログなど、時間とともに変化するデータを格納。

## パスと記号の定義
- **絶対パス**: ルート `/` から記述する「住所」。
- **相対パス**: 今いる場所（カレント）から記述する「住所」。
- `/`: ルートディレクトリ。
- `.`: カレントディレクトリ（今の場所）。
- `..`: 親ディレクトリ（一つ上の階層）。
- `~`: ホームディレクトリ（自分の初期位置）。

## 演習の記録
- `mkdir` でディレクトリを作成し、`cd` と `pwd` を組み合わせて現在地が正しく移動しているかを確認。
- 絶対パスと相対パスを使い分けて、目的のディレクトリへ移動する練習を実施。
- `ls -la` を使い、普段は見えない設定ファイル（隠しファイル）が `/` や `~` に存在することを確認。
